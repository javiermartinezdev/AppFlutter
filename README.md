# Aplicación con Flutter
Esta es una primera versión de una aplicación multiplataforma utilizando el framework **Flutter** y el lenguaje **Dart**.

En el código se implementan el siguiente esquema de carpetas: Screens, Widgets y las clases principales: app y main.

### Tecnologias utilizadas
- **Visual Studio**
- **Flutter**
- **Dart**

## Ejecutar el proyecto

1. Clona el repositorio utilizando el siguiente comando:<br>
`git clone <url del repositorio>`
2. Dar click derecho en la clase main.dart y abrir en terminal integrado.
3. Ejecutar el siguiente comando, para descargar las dependencias necesarias:
`git clone <url del repositorio>`
4. Ejecutar el siguiente comando, para lanzar la aplicación:
`flutter run`
5. En la terminal aparecerá la siguiente indicación:
   `Please choose one (or "q" to quit):`
   Elegir el dispositivo de su preferencia.

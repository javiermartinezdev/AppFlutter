## Aplicación con Flutter
Esta es la primera versión de una aplicación multiplataforma utilizando el framework **Flutter** y el lenguaje **Dart**.

En el código se implementan el siguiente esquema de carpetas: Screens, Widgets y las clases principales: app y main.

### Tecnologias utilizadas
- **Visual Studio**
- **Flutter**
- **Dart**

### Requisitos Previos:
Antes de ejecutar el proyecto, es necesario instalar el SDK de **.Flutter**. Puedes descargarlo e instalarlo desde el [sitio web oficial](https://docs.flutter.dev/get-started/install/windows/mobile). 

### Pasos para ejecutar el Proyecto:
1. **Abrir la Terminal**:
   - Crea una carpeta, abre la terminal de comandos para clonar el repositorio y guardar el proyecto.

2. **Clona el repositorio utilizando el siguiente comando**:
     ```bash
     git clone https://github.com/Tecnologias-Seminarios/FrontEnd-Flutter.git
     ```
3. **Navega al directorio del Proyecto**:
   - Utiliza el comando `cd` para cambiar al directorio donde se encuentra tu proyecto. Debes estar en el mismo directorio que el archivo `main.dart`.
   - Ejemplo: Si tu proyecto está en el escritorio, podrías escribir:
     ```bash
     cd Desktop/FrontEnd-Flutter
     ```

4. **Ejecuta el siguiente comando, para descargar las dependencias necesarias**:
     ```bash
     flutter pub get
     ```
     
5. **Ejecutar el siguiente comando, para lanzar la aplicación**:
    ```bash
     flutter run
     ```

6. **En la terminal aparecerán los dispositivos disponibles para ejecutar la aplcación, elige el dispositivo de tu preferencia. Puede ser algún navegador, como en el ejemplo: Edge, entonces elegimos el 2**:<br><br>

7. **Para finalizar la ejecución de la aplicación, solo debemos de presionar la tecla q**.
